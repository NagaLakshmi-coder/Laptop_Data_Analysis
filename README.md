Laptop Data Analysis Project

This project performs exploratory data analysis (EDA) on a dataset containing specifications of 3,976 laptops from different brands. 
The dataset includes features such as brand, processor type, RAM, storage, GPU, display, battery life, and price.

The goal of this project is to analyze how different laptop specifications influence pricing and to identify trends among laptop brands and configurations.

Dataset Description

The dataset contains detailed specifications of laptops with the following columns:

1. Brand
2. Name
3. Price
4. Processor_Name
5. Processor_Brand
6. RAM
7. RAM_TYPE
8. RAM_Expandable
9. Ghz
10. Display_Type
11. Display
12. GPU
13. GPU_Brand
14. SSD
15. HDD
16. Adapter
17. Battery_Life

This data helps analyze relationships between hardware specifications and laptop prices.

Key Insights

Some important insights discovered during the analysis:

Popular laptop brands dominate the dataset.

Higher RAM configurations tend to increase laptop prices.

Processor brand significantly influences pricing.

Laptops with dedicated GPUs are generally more expensive.

Storage configuration also impacts price variations.
